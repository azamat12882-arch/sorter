# sorter
### RU
*Эта утилита позволяет сортировать файлы по папкам.*
*Для работы утилиты распакуйте архив в удобное место, в файле config.txt укажите путь сортировки.*
*В этот путь скопируйте папки из архива.*
***ВНИМАНИЕ!!! утилита пока что не поддерживает кириллицу в путях, не распаковывайте архив в папку для сортировки.***
### EN
*This utility allows you to sort files into folders.*
*To use the utility, unzip the archive to a convenient location and specify the sorting path in the config.txt file.*
*Copy the folders from the archive to this path.*
***ATTENTION!!! The utility does not currently support Cyrillic characters in paths; do not unzip the archive to a folder for sorting.***
